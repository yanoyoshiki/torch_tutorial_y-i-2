このリポジトリは課題２に関するリポジトリです. 


# Quick start
任意の場所にて
```
git clone https://github.com/yanoyoshiki/torch_tutorial_Y-I.git

cd torch_tutorial_Y-I
cd docker

bash build.sh
bash run.sh
```

以降は
```
docker exec -it torch_tutorial_y-i　bash
```
でコンテナに接続できる

controll と P と Q
